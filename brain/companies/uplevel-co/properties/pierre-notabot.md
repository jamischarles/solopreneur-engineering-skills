# Pierre Notabot

## Overview
Public knowledge base / wiki about LLMs and AI, written by "Pierre" — a French robot character who insists he's "definitely human... probably."

## Risk Appetite
**Balanced** — SSG content site, easy to redeploy but public-facing.

## Repo
jamischarles/pierre-notabot (also aliased as pnab/)

## URL
Live (Next.js 15 SSG site)

## Purpose
Approachable AI/LLM education with personality. Uses progressive disclosure (beginner / intermediate / pro layers via accordions) so readers learn at their own level. The character makes dry technical content more engaging.

## Target Audience
- Developers learning Claude / LLM concepts
- AI-curious people from beginner to pro level
- People who learn better from personality-driven content vs. dry docs

## Positioning
Character-driven education. Pierre's French-robot persona (navy/red/cyan/gold branding) differentiates from generic AI tutorials. The progressive disclosure format is a real UX differentiator.

## Monetization
Free articles as primary content. Premium articles referenced in structure but not yet implemented. Potential for paid deep-dives or courses.

## Current State
**Pipeline: Launch & Validate > Public Page**

Site structure is live, articles have draft/review/live status workflow. Content pipeline exists but library is still being built out. Public-facing page is up — people can discover it and understand what it does.

## Growth / Marketing
- SEO-driven (AI/LLM tutorials are high-demand search terms)
- Character creates brand recognition and shareability
- Could cross-promote Save My Brain properties to AI-interested audience
